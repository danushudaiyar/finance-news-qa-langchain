# Finance News QA Tool using LangChain & OpenAI

An end-to-end AI-powered news research tool for finance that leverages **LangChain**, **OpenAI API**, and **Streamlit** to help analysts and researchers quickly extract insights from financial news articles.

## Features
- Load financial news articles from URLs
- Split text for efficient processing
- Create FAISS vector index for semantic search
- Ask questions and get answers with sources
- Streamlit interface for interactive exploration

## Tech Stack
- Python 3.x
- [LangChain](https://www.langchain.com/)
- [OpenAI](https://platform.openai.com/)
- [Streamlit](https://streamlit.io/)
- FAISS vector database
